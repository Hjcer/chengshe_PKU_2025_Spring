words.txt里面有一些单词。对于这些单词，到微软必应词典 https://cn.bing.com/dict  进行查找

有的单词能找到同义词，则把有同义词的单词及其同义词，放到一个文件

result.txt 里面，格式如下

$new
novel
newfangled
original
latest
recent
$body
form
figure
frame
physique
corpse


有的单词有图片，比如new 就有图片。
对于有图片的单词 x ，就下载一个图片文件，命名为 x.png,存到.py文件所在的文件夹下

提示，html文档里面的各种标记，往往都是成对出现的，比如： <a ....> ..... </a>, <div ....> ..... </div>


完成步骤：
1. 在 https://cn.bing.com/dict 输入一个单词，查出结果
2. 拷贝浏览器地址栏的信息(即网址，url)，把里面的单词换成别的单词，再输入到浏览器地址栏，看能否查到结果
3. 在浏览器里，输入网址 https://cn.bing.com/dict 查一个有同义词和图片的单词，比如new，然后在浏览器里查看这个网页的源码，在源码里查找一个同义词，看看同义词会被包含在什么样的唯一的模式串里面（别处没这种模式）。看的时候要往前往后多看一些
4. 在程序用  html = getHTMLText(url)就能得到一个url(网址）对应的网页。html是个字符串。
本程序要用到的url，就来自前面2中的url，把单词部分换成不同的单词。
5. 用正则表达式从html里面抽取出包含同义词的模式串。如果要一次抽取同义词，正则表达式不好写，则可以先写个简单的，抽取包含多个同义词的一个子串S，再写另一个正则表达式，去抽取S中的同
义词。

图片处理：

鼠标右键点击图片，拷贝链接。然后在网页的源码里面查找这个链接，观察出包含这个链接的模式。
然后用正则表达式抽取链接，并读出图片保存（见讲义百度图片示例）
